# 🐧 Tutorial TenRusl Linux

## Ikhtisar
Tutorial TenRusl Linux adalah perpustakaan pembelajaran terstruktur untuk administrasi Linux, backup, keamanan, virtualisasi, hosting web, dan workstation Ubuntu. Tree bahasa Inggris di bawah `assets/content/en/` adalah sumber katalog untuk proyek dan diatur sehingga Anda dapat beralih dari konsep ke pelaksanaan tugas, lalu kembali ke materi referensi ketika Anda membutuhkan jawaban cepat.

## Siapa yang Harus Menggunakan Bagian Ini
- Linux pemula yang menginginkan jalur terpandu, bukan postingan blog yang terisolasi.
- VPS dan operator server yang memerlukan daftar periksa operasional yang dapat diulang.
- Pengguna hosting web yang memelihara proxy terbalik, panel, TLS, dan backups.
- Operator virtualisasi yang mengelola penyimpanan, jaringan, templates, dan pemulihan.
- Ubuntu pengguna stasiun kerja yang menginginkan jalur bersih dari pengaturan desktop minimal hingga lebih lengkap.

## Prasyarat
- Navigasi shell dasar dengan `cd`, `ls`, `cp`, `mv`, dan `sudo`.
- Sistem pengujian, lab VM, atau server sekali pakai untuk tugas praktis.
- Kesediaan untuk memverifikasi setiap perubahan dengan logs, status service, dan pemeriksaan jaringan.

## Jalur Pembelajaran
1. Mulai dengan [server](./server/index.md) jika Anda memerlukan baseline host Linux.
2. Pindah ke [Keamanan](./security/index.md) setelah host dapat dijangkau dan di-patch.
3. Tambahkan [Backup](./backup/index.md) sebelum Anda mempercayai sistem dengan data nyata.
4. Gunakan [Virtualisasi](./virtualization/index.md) saat Anda membangun lingkungan berkerumun atau multi-VM.
5. Gunakan [workstation](./workstation/index.md) untuk pengaturan desktop dan laptop Ubuntu.

## Apa yang Harus Dibaca Terlebih Dahulu
- Admin server baru: [server](./server/index.md) lalu [Keamanan](./security/index.md).
- Operator hosting web: [server](./server/index.md), [Keamanan](./security/index.md), lalu [Backup](./backup/index.md).
- Pembuat lab atau homelab: [Virtualisasi](./virtualization/index.md) lalu [Backup](./backup/index.md).
- Pengguna desktop: [workstation](./workstation/index.md).
