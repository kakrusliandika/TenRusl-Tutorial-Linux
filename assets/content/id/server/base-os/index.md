# 📦 OS Dasar

## Ringkasan
Modul OS dasar membuat server Linux yang dapat diprediksi sebelum layanan web, panel, proxy terbalik, atau platform aplikasi ditempatkan di atasnya. Host keluarga Debian, Ubuntu, dan RHEL berbagi kebiasaan server inti, namun mereka tidak berbagi manajer paket, alat pembaruan, alur kerja jaringan, atau default layanan.

## Untuk Siapa Bagian Ini
- Operator mempersiapkan server Linux baru untuk penggunaan produksi atau laboratorium.
- Tim menstandardisasi tugas boot pertama dan rutinitas validasi.
- Pelajar yang membutuhkan dasar khusus distro sebelum berpindah ke modul khusus beban kerja.

## Prasyarat
- Akses konsol atau penyelamatan untuk rollback jaringan dan SSH.
- Paket nama host, ekspektasi DNS, model IP, dan tata letak penyimpanan.
- Kesediaan untuk memvalidasi setiap perubahan bidang kendali dari shell.

## Jalur Pembelajaran
1. Pilih distro yang sebenarnya Anda jalankan: [🖥️ Debian](./debian/index.md), [🖥️ Ubuntu](./ubuntu/index.md), atau [🖥️ RHEL-family](./rhel/index.md).
2. Baca halaman konsep terlebih dahulu sehingga kebijakan dasar jelas sebelum Anda mengubah paket atau jaringan.
3. Terapkan halaman petunjuk terfokus untuk SSH, firewall, pembaruan, penyimpanan, dan jaringan statis.
4. Ikuti tutorial secara berurutan jika Anda ingin membangun garis dasar menyeluruh secara menyeluruh.
