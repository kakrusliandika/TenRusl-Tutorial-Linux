# 📋 Glosarium

## Tujuan
Gunakan glosarium ini untuk menjaga ketentuan instalasi workstation tetap jelas dan stabil di seluruh halaman petunjuk dan tutorial Lite.

## Materi Referensi Terstruktur
| Istilah | Arti |
|---|---|
| UEFI | Mode boot firmware modern yang menggunakan Partisi Sistem EFI dan entri boot firmware |
| BIOS / Warisan | Model boot lama yang biasanya mengandalkan instalasi GRUB bertarget MBR atau BIOS |
| ESP | Partisi Sistem EFI; Partisi boot berformat FAT yang digunakan oleh sistem UEFI |
| Boot Aman | Fitur firmware yang membatasi komponen dan modul boot yang ditandatangani dapat dimuat |
| BitLocker | Fitur enkripsi disk Windows yang dapat mempersulit dual boot atau perubahan partisi |
| Intel RST / VMD | Mode firmware atau pengontrol yang dapat menyembunyikan disk dari installer Linux hingga teratasi |
| Minimal pemasangan | Pilihan penginstal yang mencoba mendapatkan tapak desktop yang lebih kecil |
| Paket Meta | Paket yang terutama bergantung pada sekelompok besar paket lainnya |
| initramf | Gambar boot awal dimuat sebelum sistem file root dipasang sepenuhnya |
| chroot | Perubahan lingkungan root sementara yang digunakan selama pemulihan dan perbaikan GRUB |

## Perintah / Potongan Pemeriksaan
Gunakan halaman petunjuk operasional terkait saat Anda memerlukan runbook yang tepat.

## Catatan Praktis
- Kata-kata UI penginstal yang tepat dapat bervariasi berdasarkan rilis, namun istilah ini tetap berguna secara operasional.
- Jika tutorial menggunakan istilah yang tidak dapat Anda jelaskan, jeda dan hapus istilah tersebut sebelum melanjutkan pembuatan.
