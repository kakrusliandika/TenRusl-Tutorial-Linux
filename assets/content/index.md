# TenRusl Tutorial Linux

Selamat datang di **TenRusl Tutorial Linux**.

Halaman ini menjadi **pintu masuk utama** untuk seluruh dokumentasi dan tutorial Linux di proyek ini. Root aplikasi diarahkan ke file ini agar saat situs dibuka, pengguna langsung masuk ke katalog dokumentasi, bukan ke contoh Markdown lama.

## Fokus utama

Dokumentasi ini disusun untuk belajar dan praktik yang lebih terarah pada beberapa area inti:

-   **Server** — fondasi sistem, reverse proxy, hosting panels, dan PaaS.
-   **Virtualization** — hypervisor, storage, jaringan, template, dan operasi harian.
-   **Security** — hardening untuk server, VM, dan web.
-   **Backup** — strategi backup dan restore untuk VPS, VM, dan aplikasi web.
-   **Workstation** — panduan workstation Ubuntu untuk kebutuhan ringan dan standar.

## Cara menggunakan dokumentasi ini

-   Gunakan **menu kiri** untuk berpindah kategori, modul, dan tutorial.
-   Gunakan **breadcrumb** di atas area konten untuk melihat posisi halaman aktif.
-   Gunakan **table of contents** di sisi kanan untuk lompat ke heading pada dokumen aktif.
-   Gunakan kolom pencarian di sidebar untuk menemukan tutorial lebih cepat.

## Struktur umum konten

Setiap modul umumnya dibagi ke beberapa kelompok:

-   **Concepts** untuk pemahaman dasar.
-   **How-to** untuk langkah cepat yang fokus pada satu tugas.
-   **Reference** untuk catatan teknis dan checklist.
-   **Tutorials** untuk alur belajar bertahap dari awal sampai selesai.

## Rekomendasi jalur baca

1. Masuk ke kategori yang sesuai.
2. Buka halaman indeks modul.
3. Mulai dari **Tutorials** untuk jalur belajar utama.
4. Gunakan **How-to** saat perlu tugas spesifik.
5. Gunakan **Reference** saat perlu checklist atau acuan teknis.
