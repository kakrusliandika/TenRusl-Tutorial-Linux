Privacy Policy

Last updated: 2026-03-19

TenRusl-Tutorial-Linux is a client-side Markdown viewer. By default, Markdown files are processed locally in your browser.

Summary

  • Your Markdown content is rendered in the browser by default.
  • Opening a local Markdown file does not upload that file to our server by default.
  • The app may store limited local preferences such as theme, language, and cached assets for offline use.

How content is processed

When you open a Markdown file, the content is parsed and rendered on your device. If you load a default example file from the site, that file is fetched like a normal web asset.

Local storage and offline cache

We may use localStorage for preferences such as theme and UI language. A service worker may cache app assets and example files to improve repeat visits and offline fallback.

Clipboard actions

If you click copy actions, the app asks the browser to write selected text to your clipboard. Clipboard behavior depends on browser permissions and secure-context requirements.

Third-party libraries and CDNs

The app may load front-end libraries such as Markdown parsing, syntax highlighting, icons, or diagram rendering from bundled assets or allowed CDNs. Those requests follow the browser’s normal network behavior.

Data sale

We do not describe this app as selling your personal data.

Changes to this policy

We may revise this policy when the app’s features or deployment model change.
