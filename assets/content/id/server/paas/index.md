# 🚀 PaaS

## Ringkasan
Bagian PaaS mencakup platform aplikasi yang di-host sendiri yang berjalan di atas infrastruktur server Linux. Fokusnya lebih pada operasional daripada pemasaran: kepercayaan repositori, visibilitas kontainer, perutean domain, cakupan pencadangan, peta layanan, dan pemulihan terkontrol ketika platform atau beban kerja mengalami gangguan.

## Untuk Siapa Bagian Ini
- Operator membangun pengiriman aplikasi yang di-host sendiri di server Linux mereka sendiri.
- Tim mengevaluasi alur kerja penerapan berbasis platform terhadap administrasi shell langsung.
- Admin yang membutuhkan cara yang peka terhadap host untuk mengaudit perubahan platform.

## Prasyarat
- Server dengan ukuran CPU, RAM, dan penyimpanan untuk beban kerja kontainer.
- Akses SSH, kontrol DNS, dan akses jaringan keluar.
- Pemahaman dasar tentang container, penyedia Git, dan perilaku proksi terbalik.
