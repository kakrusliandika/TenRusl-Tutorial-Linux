# 🐧 TenRusl Tutorial Linux

## Overview
TenRusl Tutorial Linux is a structured learning library for Linux administration, backup, security, virtualization, web hosting, and Ubuntu workstations. The English tree under `assets/content/en/` is the source catalog for the project and is organized so you can move from concepts to task execution, then back to reference material when you need quick answers.

## Who Should Use This Section
- Linux beginners who want a guided path instead of isolated blog posts.
- VPS and server operators who need repeatable operational checklists.
- Web hosting users who maintain reverse proxies, panels, TLS, and backups.
- Virtualization operators who manage storage, networking, templates, and recovery.
- Ubuntu workstation users who want a clean path from minimal to fuller desktop setups.

## Prerequisites
- Basic shell navigation with `cd`, `ls`, `cp`, `mv`, and `sudo`.
- A test system, lab VM, or disposable server for hands-on tasks.
- Willingness to verify every change with logs, service status, and network checks.

## Learning Path
1. Start with [Server](./server/index.md) if you need a Linux host baseline.
2. Move to [Security](./security/index.md) once the host is reachable and patched.
3. Add [Backup](./backup/index.md) before you trust the system with real data.
4. Use [Virtualization](./virtualization/index.md) when you are building clustered or multi-VM environments.
5. Use [Workstation](./workstation/index.md) for Ubuntu desktop and laptop setups.

## What To Read First
- New server admin: [Server](./server/index.md) then [Security](./security/index.md).
- Web hosting operator: [Server](./server/index.md), [Security](./security/index.md), then [Backup](./backup/index.md).
- Lab or homelab builder: [Virtualization](./virtualization/index.md) then [Backup](./backup/index.md).
- Desktop user: [Workstation](./workstation/index.md).
